Spellfire

A blazingly fast MMO game server

View Demo

Frontend Features

  • Rust Memory safety + high performance = code that never segfaults and always runs fast.
  • Bevy Modular ECS game engine designed for scalability, letting us build MMO-sized worlds with elegance.
  • Integrated debugging inspector Live-inspection of entities and components for rapid iteration.
  • GPU-accelerated particles using Hanabi Fire, smoke, and spell effects offloaded to the GPU, freeing up CPU for gameplay logic.

Backend Features

  • Golang Goroutines + channels = lightweight, scalable concurrency built for MMOs.
  • Autovectorized ECS Cache-friendly, SIMD-accelerated updates that chew through thousands of entities per tick.
  • sqlc database Generates type-safe SQL code at compile time, catching bugs before runtime.
